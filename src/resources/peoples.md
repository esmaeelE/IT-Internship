# Key Figures in Computing

People every IT professional should know. Not just names — understand what they did and why it matters.

---

## Dennis Ritchie (dmr)

- **Created**: [C language](https://en.wikipedia.org/wiki/C_(programming_language)) (1972), co-created [Unix](https://en.wikipedia.org/wiki/Unix) (1971)
- **At**: Bell Labs
- **Why it matters**: C is the foundation of almost every language and OS in use today. Unix shaped how all modern operating systems work.
- **Read**: [The C Programming Language](https://en.wikipedia.org/wiki/The_C_Programming_Language) (K&R) — the book that defined an era
- **Quote**: *"The only real way to write portable code is to write nothing at all that isn't portable."*

## Ken Thompson

- **Created**: Unix (1971, with Ritchie), [Go language](https://en.wikipedia.org/wiki/Go_(programming_language)) (2009, with Rob Pike)
- **At**: Bell Labs, Google
- **Why it matters**: Designed the philosophy of "do one thing and do it well" — Unix pipes, small tools, composition. Go revived simplicity in systems programming.
- **Quote**: *"When in doubt, use brute force."*

## Brian Kernighan (bwk)

- **Created**: AWK language (with Aho and Weinberger), co-authored *The C Programming Language* (K&R)
- **At**: Bell Labs, Princeton University
- **Why it matters**: Made C accessible to the world through K&R. AWK is still used daily in Unix text processing.
- **Read**: [The C Programming Language](https://en.wikipedia.org/wiki/The_C_Programming_Language) — the most influential programming book ever

## Richard Stallman (rms)

- **Created**: [GNU Project](https://www.gnu.org/), [Emacs](https://www.gnu.org/software/emacs/), [GCC](https://gcc.gnu.org/), founded [Free Software Foundation](https://www.fsf.org/) (1985)
- **Why it matters**: Without GNU, there would be no Linux as we know it. Free software movement changed how the world builds and shares code.
- **Read**: [The GNU Manifesto](https://www.gnu.org/gnu/manifesto.html)

## Linus Torvalds

- **Created**: [Linux kernel](https://github.com/torvalds/linux) (1991), [Git](https://git-scm.com/) (2005)
- **Why it matters**: Linux runs most servers, all Android phones, and all supercomputers. Git is the standard for version control worldwide.
- **Read**: [Just for Fun](https://en.wikipedia.org/wiki/Just_for_Fun) — his autobiography

## Doug McIlroy

- **Invented**: [Unix pipes](https://en.wikipedia.org/wiki/Pipeline_(Unix)) — the `|` operator
- **At**: Bell Labs
- **Why it matters**: Pipes are the foundation of Unix composability. `cat file | grep word | sort | uniq` — that's his idea.

## Donald Knuth

- **Created**: [TeX](https://www.tug.org/), [*The Art of Computer Programming*](https://en.wikipedia.org/wiki/The_Art_of_Computer_Programming) (multi-volume)
- **Why it matters**: Defined algorithm analysis. TAOCP is the deepest work on algorithms ever written. TeX is still the gold standard for typesetting.
- **Quote**: *"Premature optimization is the root of all evil."*

## Grace Hopper

- **Created**: First compiler (A-0, 1952), contributed to COBOL
- **Why it matters**: Proved that programming could use English-like syntax. Her work on compilers made programming accessible to non-mathematicians.

## Alan Turing

- **Proved**: [Universal Turing Machine](https://en.wikipedia.org/wiki/Turing_machine) — the theoretical foundation of all computers
- **Why it matters**: Before Turing, "computer" was a job title. His 1936 paper defined what computation is and what is computable. Every algorithm runs on his model.
