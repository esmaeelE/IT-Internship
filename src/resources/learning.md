## English Resources

### Linux

- [The Linux Command Line](https://linuxcommand.org/tlcl.php) — free book by William Shotts
- [Ryan's Linux Tutorial](https://ryanstutorials.net/linuxtutorial/) — beginner-friendly, 13 sections
- [Linux Journey](https://linuxjourney.com) — structured course (may be offline)

### Python

- [Python Basics](https://pythonbasics.org) — practical tutorials
- [Automate the Boring Stuff](https://automatetheboringstuff.com) — free, project-based
- [Python Official Tutorial](https://docs.python.org/3/tutorial/) — authoritative reference
- [Learn Python the Hard Way](https://learnpythonthehardway.org) — exercise-driven

### C

- [Beej's Guide to C](https://beej.us/guide/bgc/) — free, comprehensive
- [C Programming Language (K&R)](https://en.wikipedia.org/wiki/The_C_Programming_Language) — the classic

### Git

- [Pro Git Book](https://git-scm.com/book/en/v2) — free, complete reference
- [Oh Shit, Git!?!](https://ohshitgit.com) — beginner-friendly, fixes common mistakes

### Networking

- [Computer Networking: A Top-Down Approach](https://gaia.cs.umass.edu/kurose_ross/online_lectures.htm) — free lectures

### SQL

- [SQLBolt](https://sqlbolt.com) — interactive SQL exercises
- [SQLZoo](https://sqlzoo.net) — more interactive SQL practice

### General

- [Hacker News](https://news.ycombinator.com) — tech news and discussions
- [Stack Overflow](https://stackoverflow.com) — programming Q&A
- [freeCodeCamp](https://www.freecodecamp.org) — free coding courses
