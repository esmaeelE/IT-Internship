# IT Internship

Learning roadmap and resources for IT interns.

## How to Use This Book

This book is structured as a progressive learning path:

1. **[Week 1 Plan](./getting-started.md)** — Concrete steps to start today
2. **[Skills Roadmap](./roadmap/skills.md)** — Core IT skills with learning links
3. **[Philosophy](./roadmap/philosophy.md)** — Mindset and job level progression
4. **[Exercises](./exercises/exercises.md)** — Practice projects to build experience
5. **[Resources](./resources/learning.md)** — Learning materials in Persian and English
6. **[Notes](./notes/notes.md)** — Daily note-taking structure

## Principles

- Learn how to learn — that's the real skill
- Break big problems into small pieces
- Write it down — daily notes, push to repo
- Give back to the community, don't just take
