# Week 1 Plan

Concrete steps for your first week. Do them in order. Check them off.

## Day 1 — Setup

- [ ] Install a Linux distro (Ubuntu or Fedora) — dual boot or VM
- [ ] Learn to open a terminal and navigate directories (`cd`, `ls`, `pwd`)
- [ ] Install Git: `sudo apt install git` (or equivalent)
- [ ] Create a GitHub account if you don't have one
- [ ] Set up Git: `git config --global user.name "Your Name"` and `git config --global user.email "you@email.com"`

## Day 2 — Terminal Basics

- [ ] Read: [The Linux Command Line](https://linuxcommand.org/tlcl.php) — Chapter 1–3
- [ ] Practice: `cat`, `cp`, `mv`, `rm`, `mkdir`, `touch`
- [ ] Learn: `man ls`, `man cat` — how to read man pages
- [ ] Type for 20 min on [MonkeyType](https://monkeytype.com) — aim for accuracy first, speed later

## Day 3 — Text Editing & Files

- [ ] Learn basics of `nano` or `vim` (start with nano, graduate to vim)
- [ ] Read about file permissions: `ls -la`, `chmod`, `chown`
- [ ] Practice: create files, edit them, move them, delete them — all from terminal
- [ ] Write a `hello.py` in Python: `print("Hello, World!")` → run with `python3 hello.py`

## Day 4 — Python Basics

- [ ] Read: [Python Basics](https://pythonbasics.org) — Variables, Strings, Lists
- [ ] Practice: write a script that asks for your name and prints a greeting
- [ ] Learn: `if`, `for`, `while`, functions
- [ ] Push your first repo to GitHub: `git init`, `git add .`, `git commit`, `git push`

## Day 5 — Search & Documentation

- [ ] Read: [Pro Git Book](https://git-scm.com/book/en/v2) — Chapter 1 (Getting Started)
- [ ] Practice Google operators: `site:`, `filetype:`, `-`
- [ ] Look up man pages for 5 commands you've used this week
- [ ] Write a short note about what you learned today — push to repo

## Day 6 — Connect & Explore

- [ ] Join a community: [Ubuntu Forums](https://ubuntuforums.org), Reddit r/linux, Discord dev servers
- [ ] Read about Open Source: what is a pull request? how do people contribute?
- [ ] Explore [Ryan's Linux Tutorial](https://ryanstutorials.net/linuxtutorial/) — sections 1–5
- [ ] Continue typing practice — 15 min

## Day 7 — Review & Plan

- [ ] Review all notes from this week
- [ ] Review your Git log: `git log --oneline` — see your progress
- [ ] Write a weekly reflection: what went well, what was hard, what's next
- [ ] Plan Week 2: pick one topic from the [Skills Roadmap](./roadmap/skills.md) to go deeper
