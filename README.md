# IT Internship

Learning roadmap and resources for IT interns.

## Book

Read the book online or build it locally with [mdBook](https://rust-lang.github.io/mdBook/):

```sh
# install mdBook
cargo install mdbook

# serve with live reload
mdbook serve

# or build static HTML
mdbook build
```

Output goes to `book/`.
