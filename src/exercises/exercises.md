# Practice Projects

Start from the top, work your way down. Each project teaches different skills.

## Beginner

**Tic-Tac-Toe (XO)** — Python basics, loops, conditionals, user input
  - CLI version first, then optional: Pygame GUI
  - Skills: `if/else`, `for/while`, 2D lists, functions

**Snake Game** — Game loop, input handling, collision detection
  - Use Pygame or terminal-based
  - Skills: arrays, game state, coordinates

**Search & Sort Visualizer** — Implement bubble sort, selection sort, binary search
  - Print array state at each step
  - Skills: algorithms, loops, arrays, time complexity awareness

## Intermediate

**Web Crawler** — Fetch web pages, extract links, follow them
  - Use `requests` + `BeautifulSoup` (Python) or Scrapy
  - Skills: HTTP, HTML parsing, recursion, rate limiting, data storage

**RESTful API** — Build a simple API (todos, books, contacts)
  - Use Flask or FastAPI (Python)
  - Skills: HTTP methods (GET/POST/PUT/DELETE), JSON, status codes, SQLite

**Message Queue Demo** — Producer/consumer pattern with RabbitMQ
  - Send and receive messages between two scripts
  - Skills: async programming, message queues, distributed systems basics

## Advanced

**AI/ML Intro** — Train a simple model (classifier or regression)
  - Use scikit-learn: iris dataset or MNIST
  - Skills: data preprocessing, train/test split, evaluation metrics

---

> **Tip**: Don't skip the beginner projects. They build muscle memory. Speed comes from repetition, not from jumping ahead.
