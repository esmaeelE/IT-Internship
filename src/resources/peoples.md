# Key Figures in Computing

- **Ken Thompson** — co-created Unix, designed Go
- **Dennis MacAlistair Ritchie (dmr)** — created C language, co-created Unix
- **Richard Matthew Stallman (rms)** — founded GNU Project, Free Software Foundation, created Emacs
- **Linus Torvalds** — created Linux kernel, created Git
- **Doug McIlroy** — invented Unix pipes, contributed to early programming
- **Brian W. Kernighan (bwk)** — co-authored "The C Programming Language" (K&R)
